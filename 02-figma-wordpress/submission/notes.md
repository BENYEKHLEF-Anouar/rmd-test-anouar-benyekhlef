## WordPress Theme

A custom WordPress theme was created specifically for this project.

The website styling and layout were hand-coded using:

* HTML
* CSS

The custom design files were added directly into the WordPress theme directory and integrated into the site structure.

The theme was developed locally using [LocalWP](https://localwp.com/#) before deployment to Wasmer hosting.
